Code examples
-------------
This page includes code samples that you can try out in Ruby or Unix shells. See
[this page]({relocatable: /basics/code_examples.html}) for explanations on the notation.
{: .warning}

Moreover, to run the examples on this page, you must first set up your ruby
shell:

     $ irb
{coderay:: ruby}
 >> require 'roby/standalone'
 >> include Roby
 >> plan = Roby.plan
{coderay}
